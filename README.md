# Sudoku solver
Sudoku solver 
representing sudoku board:
```
[[x00, x01, x02, x03... x08],
 [x10, x11, x12, x13... x18],
 ...
 [x80, x81, x82, x83... x88]]
```
Values correspond to the value at the rth row
Values can be empty

Example,
```
[[-1,  1,  5, ...],
 [-1, -1, -1, ...],
 [ 6, -1, -1, ...]
 ...]
```

board example:
```
 -----------
|     1   5 | ...
|           | ...
| 6         | ...
 -----------
 ...
```
The goal is to solve sudoku Python
